# Estructura esperada de Google Sheets

Hojas relevantes:
- ADMINISTRACIÓN
- FITNESS
- SALON
- AULA
- LOCAL 1

Columnas base de inscriptos:
- AB: Id
- AD: Fecha
- AF: Nombre
- AI: Apellido
- AL: DNI
- AN: Teléfono
- AP: Actividad
- AR: Modalidad
- AT: Cuota
- AV: Estado
- AY: Instructor / Profesor

En el MVP se prioriza el campo **Estado = Adeudando**.
