# Política de uso de WhatsApp

## Fase 1
- El sistema usa enlaces `wa.me` con mensaje prearmado.
- El envío lo confirma manualmente el usuario desde WhatsApp Web.
- No hay automatización invasiva.
- Debe evitarse spam o contactos no autorizados.

## Fase 2 (objetivo)
- Migrar a WhatsApp Business Platform / Cloud API.
- Usar plantillas aprobadas y trazabilidad de entrega.
